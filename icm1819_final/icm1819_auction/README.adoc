== Modo de Adjudicação (Metodo de Premiação)

Metodo de ganhar o leilão 

* 1) Conceder o ganho ao primeiro lance com valor >= VALOR_MINIMO
* 2) Apos FIM_LICITACAO, ganha que o melhor lance (deve ser >= VALOR_MINIMO)
